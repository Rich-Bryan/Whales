# Smart_Flow

## Need to have a env 

### Mac
```
python3 -m venv .venv
```

### Window
```
py -3 -m venv .venv
```

## Activate the environment 

### Mac
```
. .venv/bin/activate
```
### Window 
```
.venv\Scripts\activate
```

## need to make a config.py file for your API keys

```python
API_KEYS = "Your API keys"
```
